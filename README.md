for public consumption.

## Documentation

[https://django-crawfish.readthedocs.io/en/latest](https://django-crawfish.readthedocs.io/en/latest)

### Installation

```bash
pip install django-crawfish
```

## Development

Install package dependencies:

```bash
pip install -e .
```

Install dev dependencies:

```bash
pip install -e '.[dev]'
```

Run tests:

```bash
python runtests.py

```
